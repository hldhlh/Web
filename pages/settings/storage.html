S3 设置

通过 S3 协议连接到 Supabase Storage，您可以使用任何与 S3 兼容的客户端或工具。 请注意，密钥信息仅保存在当前会话中，刷新页面后需要重新输入。

密钥 ID 由系统自动配置，无需手动输入
出于安全考虑，密钥仅保存在当前会话中，不会持久存储

使用说明

  1. 从 Supabase 控制台获取您的 S3 密钥（Secret Access Key）
  2. 在上方表单中输入您的密钥并保存
  3. 使用"测试 S3 连接"按钮验证连接是否正常
  4. 现在您可以使用任何支持 S3 协议的客户端或工具访问您的存储桶

S3 客户端配置参数：
端点: 加载中...
区域: 加载中...
访问密钥 ID: 从 Supabase 控制台获取
密钥: 从 Supabase 控制台获取
存储桶: cloud-storage
